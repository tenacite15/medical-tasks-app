# medical-tasks-app

Application full-stack de gestion des tâches médicales

# Frontend (React + TypeScript)
- Interface moderne et responsive
- Tableau interactif avec tri sur les colonnes
- Filtres multiples en temps réel

# Backend (Node.js)
- API REST complète pour gérer 10 000 tâches médicales
- Endpoints CRUD
- Filtrage avancé par statut, priorité, catégorie, médecin assigné
- Recherche textuelle
